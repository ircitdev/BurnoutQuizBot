# BurnoutQuiz Bot

Telegram-бот с квизом «Готовы ли вы к Тихой Японии?» для проекта Burnout Bootcamp.

## Возможности

- **Квиз из 10 вопросов** с картинками, пояснениями и подсчётом баллов
- **Лид-магнит** — автоматическая выдача PDF-гайда по результатам
- **SQLite база данных** — хранение пользователей и ответов
- **Диплинки** — отслеживание источника перехода (`?start=source`)
- **Форум-топики** — автоматическое создание топика в супергруппе для каждого пользователя с результатами квиза
- **Двусторонний чат** — переписка с пользователем через топик в группе
- **Рассылка** — команда `/broadcast` для массовой отправки сообщений всем пользователям

## Требования

- Python 3.10+
- python-telegram-bot 20.7

## Установка

```bash
git clone https://github.com/ircitdev/BurnoutQuizBot.git
cd BurnoutQuizBot
pip install -r requirements.txt
```

## Настройка

В файле `bot.py` задайте:

- `TOKEN` — токен бота от @BotFather
- `ADMIN_IDS` — список Telegram ID администраторов
- `GROUP_ID` — ID супергруппы с форум-топиками

## Запуск

```bash
python bot.py
```

## Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Начать квиз |
| `/start <source>` | Начать квиз с отслеживанием источника |
| `/broadcast` | Начать рассылку (только для админов) |

## Структура

```
bot.py              # Основной файл бота
requirements.txt    # Зависимости
bot_database.db     # SQLite база (создаётся автоматически)
```
